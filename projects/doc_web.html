Tentang Nexus

Nexus adalah platform revolusioner yang menggabungkan teknologi blockchain dan kecerdasan buatan untuk menjaga memori keluarga secara digital. Setiap keluarga memiliki Soulbound Token (SBT) unik yang menjadi identitas digital mereka di blockchain.

Fitur Utama

  • Digital Identity: Soulbound Token sebagai identitas keluarga di blockchain
  • Memory Preservation: Penyimpanan memori keluarga sebagai NFT di blockchain
  • AI Guardian: Kecerdasan buatan yang memahami konteks keluarga
  • Digital Will: Sistem wasiat digital untuk warisan keluarga
  • Decentralized Storage: Penyimpanan data di IPFS

Smart Contract

Smart Contract Soulbound Token (SBT) adalah inti dari sistem identitas digital keluarga:

Fitur Kontrak Utama

  • Minting: Membuat token baru untuk keluarga
  • Soulbound: Token tidak dapat ditransfer ke alamat lain
  • Metadata: Penyimpanan URI metadata di IPFS
  • Access Control: Manajemen hak akses berbasis kepemilikan token

Kode Kontrak Inti

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@openzeppelin/contracts/token/ERC721/ERC721.sol";

contract FamilySoulbound is ERC721 {
    uint256 private _tokenIdCounter;
    address public owner;
    mapping(address => bool) public hasMinted;
    
    struct FamilyIdentity {
        string familyName;
        uint256 generation;
        address[] members;
    }
    
    mapping(uint256 => string) private _tokenURIs;

    constructor() ERC721("FamilySoulbound", "FAM-SBT") {
        owner = msg.sender;
    }

    function mintFamilySBT(string memory tokenURI) external payable {
        require(msg.value == 0.01 ether, "Incorrect ETH amount");
        require(!hasMinted[msg.sender], "Already minted");
        
        uint256 tokenId = _tokenIdCounter++;
        _mint(msg.sender, tokenId);
        _setTokenURI(tokenId, tokenURI);
        hasMinted[msg.sender] = true;
    }

    function _setTokenURI(uint256 tokenId, string memory _tokenURI) internal {
        require(_exists(tokenId), "Token doesn't exist");
        _tokenURIs[tokenId] = _tokenURI;
    }

    function tokenURI(uint256 tokenId) public view override returns (string memory) {
        require(_exists(tokenId), "Token doesn't exist");
        return _tokenURIs[tokenId];
    }

    // Fungsi untuk membuat token non-transferable (Soulbound)
    function _beforeTokenTransfer(
        address from,
        address to,
        uint256 tokenId
    ) internal override {
        require(from == address(0), "Token is soulbound and non-transferable");
        super._beforeTokenTransfer(from, to, tokenId);
    }
}

Integrasi Wallet

Contoh kode untuk menghubungkan wallet:

async function connectWallet() {
    if (typeof window.ethereum !== 'undefined') {
        try {
            const accounts = await window.ethereum.request({ 
                method: 'eth_requestAccounts' 
            });
            
            userAddress = accounts[0];
            updateUI();
            
            // Load data pengguna
            loadUserData();
            
        } catch (error) {
            console.error("User denied account access");
        }
    } else {
        alert("Please install MetaMask!");
    }
}

Interaksi dengan Smart Contract

async function mintFamilySBT() {
    const contract = new ethers.Contract(
        CONTRACT_ADDRESS, 
        CONTRACT_ABI, 
        signer
    );
    
    try {
        // Dapatkan metadata URI dari IPFS
        const metadataURI = await uploadMetadataToIPFS();
        
        // Mint SBT
        const tx = await contract.mintFamilySBT(metadataURI, {
            value: ethers.utils.parseEther("0.01")
        });
        
        await tx.wait();
        console.log("SBT minted successfully!");
        
    } catch (error) {
        console.error("Minting failed:", error);
    }
}

Sistem Kecerdasan Buatan

Nexus AI adalah asisten pribadi yang memahami konteks keluarga dan sejarah mereka:

Arsitektur AI

  • Personalization Layer: Memahami identitas keluarga spesifik
  • Memory Context: Mengakses memori keluarga dari blockchain
  • Natural Language Processing: Memproses permintaan pengguna
  • Response Generation: Membuat respons yang relevan dan personal

Contoh Implementasi AI

const NexusAI = {
    identity: null,
    
    getResponse: function(input) {
        if (!this.identity) return "Please connect wallet";
        
        input = input.toLowerCase();
        
        // Deteksi perintah lokasi
        if (input.includes("lokasi") || input.includes("dimana")) {
            if (!this.identity.locations) return "No location data";
            
            let response = "📍 Lokasi Keluarga:\n";
            for (const [member, location] of Object.entries(this.identity.locations)) {
                response += `\n• ${member}: ${location}`;
            }
            return response;
        }
        
        // Deteksi perintah memori
        if (input.includes("memori") || input.includes("kenangan")) {
            const memories = this.identity.memories;
            if (memories.length === 0) return "No memories recorded";
            
            let response = `📜 Memori Keluarga ${this.identity.familyName}:\n`;
            memories.forEach((mem, idx) => {
                response += `\n${idx+1}. ${mem.title} (${mem.date})`;
            });
            return response;
        }
        
        // Respons default
        return `Sebagai penjaga memori keluarga ${this.identity.familyName}, 
saya akan mencatat ini untuk generasi mendatang...`;
    }
};

Deployment ke IPFS

Nexus didesain untuk di-deploy sebagai aplikasi terdesentralisasi di IPFS:

Langkah Deployment

  1. Build Aplikasi: 
    npm install -g parcel-bundler
parcel build index.html --public-url ./
  2. Upload ke IPFS: 
    # Install IPFS CLI
ipfs init

# Start IPFS daemon
ipfs daemon &

# Upload build folder
ipfs add -r dist/
  3. Akses Aplikasi:

    Gunakan CID yang dihasilkan untuk mengakses aplikasi:

    https://ipfs.io/ipfs/QmXoypizjW3WknFiJnKLwHCnL72vedxjQkDDP1mXWo6uco
